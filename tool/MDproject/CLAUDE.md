# test1 - 2D 动作冒险游戏

基于 Godot 4.6 开发的类塞尔达风格 2D 俯视角动作冒险游戏。



## 项目约定

- **沟通语言**：与 AI 助手交流统一使用中文
- **文件命名**：所有文件名与目录路径使用英文，避免编码兼容问题


## 开发环境

- **引擎版本**：Godot 4.6
- **物理引擎**：Jolt Physics（3D 物理后端）
- **渲染驱动**：Direct3D 12（Windows）
- **平台**：Windows 11
